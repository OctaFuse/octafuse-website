This site provides product overview and deployment guides. **Authoritative API and migration content remains in the [octafuse-gateway](https://github.com/OctaFuse/octafuse-gateway) repository**; this site structures navigation and summaries, with links back to source docs where needed.

## Suggested reading order

1. [Quick start](/en/docs/quick-start/) — run **Proxy** + **Admin** with Docker, Cloudflare, or Node
2. [Deployment](/en/docs/deploy/) — choose Cloudflare, hybrid, or fully self-hosted
3. [Main repo doc index](https://github.com/OctaFuse/octafuse-gateway/blob/main/docs/README.md) — architecture, operations, and conventions
